How to View the Print Shop Application

Step 1: Access the Webview

In the Replit interface, look for a tab called "Webview" or a browser icon at the top of the screen.

Click on this tab to open the application preview.

Step 2: Login to the Application

Once the webview loads, you'll see a login screen.

Use these credentials to log in:

  • Username: admin
  • Password: password123

Step 3: Navigate the Print Shop System

After logging in, you can access various sections from the navbar:

  • Dashboard: Overview of recent orders and activity
  • Customers: Manage customer information
  • Orders: View and manage print orders
  • Quotes: Create and manage quotes for potential orders
  • Saved Prices: Manage pricing templates and materials
  • Paper Options: Configure available paper types
  • Finishing Options: Configure available finishing options

Tip: If the Webview tab is not showing the application, try refreshing the tab or checking the "Start application" workflow to ensure it's running properly.

Alternative: Direct URL Access

If you're having trouble with the Webview tab, you can try accessing the application directly via URL.

Look for your Replit project URL (typically something like https://your-project-name.username.repl.co) and visit it in a new browser tab.
