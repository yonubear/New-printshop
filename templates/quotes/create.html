Quote Creation Instructions

To create a new quote, follow these steps:

  1. Select a customer from the dropdown menu
  2. Enter a custom quote number or leave blank for auto-generated
  3. Enter a title and optional description for the quote
  4. Set a valid until date (optional)
  5. Click "Create Quote" to save
  6. After creating the quote, you'll be able to add items on the next screen
Important Notes
  • Quotes start in "Draft" status by default
  • Once created, you can add items with detailed specifications
  • You can always edit the quote details later
  • When the quote is complete, you can change the status to "Sent"
